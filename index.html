How does the Internet work?

Protocols and Packets

Everytime we visit a website, a request for the files required to load the website is made. This request goes through a few protocols before reaching the server.

The server then sends us the data required, usually in individual packets, and each packet will find its way to you.

What's in these packets?

HTML: The content of the website

CSS: The design of the website

JS: The behaviour of the website

Hypertext Markup Language (HTML)

Invented back in 1980 by Tim Berners-Lee for researchers to use.

Basically allows users to "tag" the content that they were sending over the Internet.

Current version: HTML5

Cascading Style Sheets (CSS)

One of the many types of style sheets invented to help style HTML files.

Allows users to style and dictate how their HTML pages should look like.

Current version: CSS3

JavaScript (JS)

Created by Netscape to enable dynamic behaviour in their browser, Navigator, back in 1995.

Allows users to add logic and behaviour to their websites.

Not related to Java!

Most widely supported version: ES5

div-span.html

<!DOCTYPE html>
<html>
    <head>
        <title>My Fifteenth Web Page!</title>
        <style>
            #top {
                font-size: 36px;
            }

            #middle {
                font-size: 24px;
            }

            #bottom {
                font-size: 12px;
            }

            .name {
                font-weight: bold;
            }
        </style>
    </head>
    <body>
        <div id="top">
            This is the <span class="name">top</span> of my web page.
        </div>
        <div id="middle">
            This is the <span class="name">middle</span> of my web page.
        </div>
        <div id="bottom">
            This is the <span class="name">bottom</span> of my web page.
        </div>
    </body>
</html>

Homework for this week

Build a single web page about something that interests you.


Requirements:

  • Your web page must include at least one list (ordered or unordered) and at least one image.
  • (Optional) Add in a table as well.
  • Your website must have a stylesheet file.
  • Your stylesheet(s) must use at least five different CSS properties. You must use the #id selector at least once, and the .class selector at least once.
